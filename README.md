

# Git Command :

# First Time :
	1. git init 
	2. git add file/folder name
	3. git commit -m “first commit”
	4. git branch -M master
        5. git remote add origin <URL>
	6. git push -u origin master
# Second Time Onwards :
	1. git status 
	2. git add file/folder name
	3. git commit -m “second commit”
	4. git push
